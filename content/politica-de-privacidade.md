# Política de Privacidade

**Última atualização:** maio de 2025

## 1. Sobre este documento

Esta Política de Privacidade descreve como o **Doce Assistente** coleta, usa e protege os dados pessoais dos usuários do sistema de gestão escolar disponível em [app.docescriancas.com](https://app.docescriancas.com).

## 2. Quais dados coletamos

- **Dados de autenticação:** nome e endereço de e-mail fornecidos via login com Google (Google OIDC)
- **Dados escolares:** informações de alunos, responsáveis, professores e funcionários inseridas pela escola
- **Dados financeiros:** informações de cobranças e pagamentos de mensalidades
- **Dados de uso:** registros de acesso e interações com o sistema para fins de funcionamento

## 3. Como usamos os dados

Os dados coletados são utilizados exclusivamente para:

- Autenticar e identificar usuários no sistema
- Gerenciar a rotina escolar (agenda, materiais, comunicados)
- Processar cobranças de mensalidades
- Melhorar o funcionamento do sistema

## 4. Compartilhamento de dados

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros para fins comerciais.

Os dados podem ser compartilhados apenas com:

- **Google:** para autenticação via Google Login
- **Asaas:** para processamento de pagamentos de mensalidades
- **Autoridades legais:** quando exigido por lei

## 5. Armazenamento e segurança

Os dados são armazenados em servidores seguros. Adotamos medidas técnicas para proteger as informações contra acesso não autorizado, alteração ou divulgação indevida.

## 6. Seus direitos

Você tem direito a:

- Acessar os dados que temos sobre você
- Solicitar a correção de dados incorretos
- Solicitar a exclusão dos seus dados
- Revogar o consentimento a qualquer momento

## 7. Contato

Para dúvidas, solicitações ou exercício dos seus direitos, entre em contato:

**E-mail:** store_nthing@proton.me

## 8. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data de última atualização estará sempre indicada no topo deste documento.
